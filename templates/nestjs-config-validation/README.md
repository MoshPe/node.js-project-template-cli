## Description

This is a [Nest](https://github.com/nestjs/nest) backend service written in TypeScript. This template contains:
- premade Swagger API static server.
- Documentation static server.
- Global service Configuration + config validation.
- Dev + Prod stages Dockerfile.
- Script `makefile` with linter.
- Restart service feature based on events.

## Installation

```bash
$ npm install
```
After installing all the dependencies, You need to go through all the following files and update Your service name:
1. [configuration.interface.ts](/templates/nestjs-config-validation/src/config/configuration.interface.ts)
2. [configuration.ts](/templates/nestjs-config-validation/src/config/configuration.ts)
3. [validate.config.ts](/templates/nestjs-config-validation/src/config/validate.config.ts)
4. [app.controller.spec.ts](/templates/nestjs-config-validation/src/controller/app.controller.spec.ts)
5. [app.service.ts](/templates/nestjs-config-validation/src/controller/app.service.ts)
6. [SERVICE_NAME.config.model.json](/templates/nestjs-config-validation/src/models/SERVICE_NAME.config.model.json)
7. [strings.service.ts](/templates/nestjs-config-validation/src/utils/strings.service.ts)
8. [app.module.ts](/templates/nestjs-config-validation/src/app.module.ts)
9. [SERVICE_NAMEService.ts](/templates/nestjs-config-validation/src/SERVICE_NAMEService.ts)
10. [main.ts](/templates/nestjs-config-validation/src/main.ts)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
